Clase 05, subiendo un proyecto a github

primeros pasos, en local

  • abrir terminal de git bush
  • situarse en la carpeta(cd)
  • crear carpeta donde estara el repo(mkdir)
  • ingresar en la carpeta creada y crear un archivo index.html(touch)
  • iniciar un repositorio (git init)

pasos en git hub

  1. ingresar en git hub
  2. crear un repositorio
  3. agregar el link del repositorio al proyecto local(git add remote)
  4. actualizar el repositorio (git add "name_file", git commit, git push)
